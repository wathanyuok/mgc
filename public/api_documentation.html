Financial Segments / Classification

Segment Cascade Overview

MA → CA → Transaction (Migration 0049/0050/0051)

Concept

ระดับSegmentField
Master AgreementSubsidiary (col 7)master_agreements.subsidiary_id · FK→subsidiaries
Credit AgreementClass / Business Type (col 17)credit_agreements.class_id · FK→classes (default to Tx)
Transaction (9 tables)Department + Location + Class override + RPTdepartment_id, location_id, class_id_override, rpt

Tx tables affected

loans, leases, promissory_notes, floor_plans, overdrafts,
trust_receipts, letters_of_credit, letter_guarantees, fx_forwards

RPT enum

ValueDescription
Externalคู่สัญญานอกกลุ่ม
In-groupคู่สัญญาในกลุ่ม MGC (Related Party)
Otherกรณีอื่น/ไม่ระบุ

All segment FKs are nullable per MoM §39 — fields เปิดให้กรอก ไม่บังคับครบ. Phase 1: local masters · Phase 2: sync NetSuite Lookup.
